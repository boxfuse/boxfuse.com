Boxfuse supports Apps written in Java or any other language and packaged as an Executable Jar file running on either OpenJDK 7.x or 8.x.

Root Certificates

By default, Boxfuse uses the same root certificate bundle as the latest version of Firefox. Additionally Boxfuse also includes the root certificates for Amazon RDS, so you can connect securely to RDS databases out of the box.

You can, however, ship your own set of root certificates, by placing them in a KeyStore inside the Jar file as /cacerts. Boxfuse will then automatically configure the JRE to use these instead.

Java Agents

If you wish to launch the JRE with one or more Java Agents, simply place the Java Agent Jar files inside the Jar file under /javaagents/. Boxfuse will then automatically configure the JRE to use these Java Agents.
