AWS

For production, Boxfuse can run your Apps on AWS using your AWS account.

Regions

Boxfuse supports the ap-northeast-1, ap-southeast-1, ap-southeast-2, eu-central-1, eu-west-1, sa-east-1, us-east-1, us-west-1 & us-west-2 regions.

We recommend the eu-central-1 region, as it is currently the only one optimized for deployment speed with new Images deployed in under 60 seconds. As Boxfuse expands its footprint, more regions will be optimized for fast deployment. The order will be based on based on customer usage and demand. This does not affect the runtime speed of your instance, which is determined by the Instance type you selected.

Instance Types

Boxfuse supports all t2, m3 and c3 Instance Types in all available regions.

App Types

Both Elastic IP (single Instance) and ELB (multiple Instances in multiple Availability Zones) apps are supported. Boxfuse always performs Zero Downtime upgrades of your Apps when you redeploy them.
