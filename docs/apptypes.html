Boxfuse offers you 3 types of apps you can use:

Type Entry point Max instances Auto-Scaling Auto recovery Usage
single-instance Elastic IP 1 no yes (with minimal downtime) Simple, non-critical applications using HTTP(S)
load-balanced Elastic Load Balancer AWS limit yes yes Robust production workloads using HTTP(S)
worker custom AWS limit yes yes Robust production workloads using queuing, time-based triggers or client-side load balancing

All apps are updated with zero-downtime blue/green deployments.
