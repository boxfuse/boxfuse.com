create

Creates a new App in the Boxfuse Console.

Usage: boxfuse create app

> boxfuse create hello

Creating axelfontaine/hello ...
Successfully created axelfontaine/hello

Arguments

Name Default Description
-apptype=... single-instance The type of app to create. Possible values: Create a single-instance (elastic IP) or a load-balanced app (ELB)
