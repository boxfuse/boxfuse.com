create

Creates a new App in the Boxfuse Console.

Usage: boxfuse create app

> boxfuse create hello

Creating myuser/hello ...
Successfully created myuser/hello

Settings

Argument boxfuse.conf property Default Description
-apptype=... apptype single-instance The type of app to create. Possible values: Create a single-instance (elastic IP) or a load-balanced app (ELB)
-dbtype=... dbtype none The type of database your app needs. Possible values: none (no database) or postgresql (Boxfuse-managed PostgreSQL database)
