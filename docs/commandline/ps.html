ps

Lists the running Instances on either your machine or AWS

Usage: boxfuse ps filter

> boxfuse ps

Running Instances on VirtualBox:
+-------------+-----------+---------------------+-----------------------+---------------------+
|  Instance   |   Image   |        Type         |          URL          |     Launched at     |
+-------------+-----------+---------------------+-----------------------+---------------------+
| vb-ec717c5e | hello:1.0 | 4 CPU / 1024 MB RAM | http://127.0.0.1:8888 | 2015-03-03 19:09:36 |
+-------------+-----------+---------------------+-----------------------+---------------------+
Total: 1

Supported filters:

You can optionally add a filter to limit results to either a specific app, a specific image or a specific instance.
Example: boxfuse ps myapp:123


Arguments:

Name Default Description
-env=dev|test|prod dev Use the AWS test or prod environments instead of the local dev one
