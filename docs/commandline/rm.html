rm

Removes this Image from your machine or the Boxfuse Vault.

Usage: boxfuse rm image

> boxfuse rm hello:1.0

Destroying Image axelfontaine/hello:1.0 ...
axelfontaine/hello:1.0 destroyed in 00:00.000s

Arguments

Name Default Description
-vault false Removes the Image from the Boxfuse Vault instead of your local machine
