open

Opens a web browser pointing at the https or http port of an Instance.

Usage: boxfuse open <instance>

> boxfuse open vb-5f3cf9387

Launching default browser to access Instance 'vb-5f3cf9387' (Url: http://127.0.0.1:8888) ...

Arguments:

Name Platforms Default Description
-platform=... virtualbox The Platform to use
Choices: virtualbox | aws
-aws.region=... aws eu-west-1 The AWS region to use
