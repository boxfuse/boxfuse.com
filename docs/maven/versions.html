boxfuse:versions

Lists all versions of a specific Component available in the Boxfuse Inventory.

Usage: mvn boxfuse:versions -Dboxfuse.component=component

> mvn boxfuse:versions -Dboxfuse.component=tomcat

Available tomcat versions
-------------------------
8.0.9
8.0.8
7.0.55
7.0.53
7.0.47
7.0.42


Properties:

Plugin Parameter Maven/System property Default Description
user boxfuse.user Required - Your Boxfuse Client user
secret boxfuse.secret Required - Your Boxfuse Client user
component boxfuse.component The Component to get the versions for

Sample Configuration

<configuration>
    <user>1234567890abcdef1234567890abcdef12345678</user>
    <secret>ABCDEFGHIJKL1234567abcdefghijklmnopqrstu</secret>
    <component>tomcat</component>
</configuration>
