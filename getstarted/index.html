Get Started with Boxfuse

 

Welcome to Boxfuse, Immutable Infrastructure made easy.

Boxfuse is the easiest, most reliable and secure way to run your JVM-based apps on AWS.

In just a few seconds Boxfuse fuses a Payload, like a Spring Boot Executable Jar, Dropwizard Executable Jar, Play Dist Zip, Tomcat War file, TomEE War file or a Jar file, together with the required Components, like Tomcat or OpenJDK into an Image we call a Bootable App.

Boxfuse then runs this Image unchanged on all environments, from Dev on VirtualBox to Prod on AWS. This can be done either via the Command-line client, the Maven Plugin or the Gradle Plugin.
